## Install

- Node.js 20+ is required.

### NPM

To install via NPM:

```
npm install --save @bedrock/recaptcha
```

### Development

To install locally (for development):

```
git clone https://github.com/digitalbazaar/bedrock-recaptcha.git
cd bedrock-recaptcha
npm install
```
